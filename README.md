# ars-coords

An easy sime FiveM script to copy coords to clipboard in different formats using ox_lib menu

Formats:
x, y, z
x y z
{x, y, z}
x = x, y = y, z = z
vector3(x, y, z)
vector4(x, y, z, w)
heading

Use /coords open 

Use /coords open the menu.

Dependency: 
ox_lib
https://github.com/overextended/ox_lib
